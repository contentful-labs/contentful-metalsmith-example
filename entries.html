The data shown on this page has been generated using the example blog space.

On this example the request is not limited. It fetches all entries.

title: Contentful Blog Example space - all entries
contentful:
  content_type: '*'
layout: index.html

The entries are available in data.entries or grouped by content type in data.contentTypes.

<!-- all entries -->
<ul>
  {{#each data.entries }}
    <li>{{sys.id}}</li>
  {{/each}}
</ul>

<!-- entries grouped by content type id -->
<ul>
  {{#each data.contentTypes.2wKn6yEnZewu2SCCkus4as }}
    <li>{{fields.title}}</li>
  {{/each}}
</ul>
